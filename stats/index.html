About

This page displays statistics for players, maps, and factions via a manual collection process. While this can never be truly comprehensive, it provides a fun and interesting overview of various aspects of the game.

All data is created and maintained by F9Bomber (Steam | YouTube), based on VSR games hosted by the BZ2 Vet Strat Discord community (yes, we're still here!).

For source code and technical details, see the Github repo.
