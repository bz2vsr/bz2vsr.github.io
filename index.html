About

This site provides real-time updates on Battlezone II: Combat Commander multiplayer games.

It prioritizes VSR games hosted by the BZ2 Vet Strat Discord community (yes, we're still here!).

It was developed using the MultiplayerSessionList API created by Nielk1 (Github | Twitter).

For source code and technical details, see the Github repo.
