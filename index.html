About

This site provides real-time updates on Battlezone II: Combat Commander multiplayer games.

It prioritizes VSR games hosted by the Battlezone and BZ2 VSR Vets Discord communities.

Credits
  • Game Data API created by Nielk1 (Github | Twitter).
  • ODF data provided by AI_Unit.
  • Game statistics provided by F9bomber.
  • Map size data provided by VTrider (Github | Steam).
  • Map scrap loose data provided by Blue Banana.

For source code and technical details, see the Github repo.
